# Frontend

This is a React + TypeScript frontend for the Job Processor system.

## Features

- **Real-time Job Updates**: Polls the backend every 2 seconds to fetch job status updates
- **Error Handling**: Gracefully handles network errors and malformed responses
- **Responsive Design**: Works on desktop and mobile devices
- **User Feedback**: Shows loading states and error messages

## Architecture

- **React 18**: Modern UI library
- **TypeScript**: Type-safe component development
- **Vite**: Fast build tool
- **CSS-in-JS**: Component styles with animations

## Development

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

```bash
npm run build
npm run preview
```
